Linux

1. ¿Qué es Linux?

Linux es un clon de el sistema operativo Unix, Linux es de codigo abierto lanzado en 1991 por Linus Torvalds escrito en el lenguaje de programación C.

2. ¿Es Linux usado?

La mayoria de paginas web que visites usaran como sistema operativo Linux o distribuciones de Linux como es Ubuntu. Android el sistema operativo mas usado del mundo, esta basado en Linux. Además Linux es usado en el entorno de programación.

3. ¿Que es el Kernel de Linux?

El Kernel de linux seria el elemento principal de los sitema operativos de Linux, la interfaz entre hardware de un ordenador y sus procesos.

Kernel, en español "Núcleo", porque se encuentra dentro del sistema operativo y controla todas las funciones principales del hardware, ya sea de un ordeandor portatil, telefono, servidor, etc..

Sus funciones son gestionar la memoria, gestionar sus procesos, controladores de dispositivos, seguridad y llamadas al sistema.
