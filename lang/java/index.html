Java

1. ¿Qué es Java?

Java es un lenguaje de programación lanzado en 1996 es un lenguaje de programación orientada a objetos (POO en español u OOP en ingles). Compilado a bytecode .class e interpretado por Assembly en la JVM. Además un programa de Java puede ejecutarse en cualquier dispositivo y sistema operativo mientras tenga Java instalado.

Java fue creado por Sun Microsystems en 1995 y adquirido por Oracle en 2010 por 7.400 millones (7,4B).

2. ¿Es Java altamente usado?

Java está instalado en MÁS de 13.000 (13B) millones de dispositivos. Esta cifra es extremadamente alta convirtiéndolo en uno de lo más lenguajes de programación usados, si no el que más.

3. ¿Es Java gratuito?

Al igual que la mayoría absoluta de lenguajes de programación Java es totalmente gratis, tanto para uso personal y uso para el desarrollo. Podrás descargarlo en https://www.java.com/download/.

4. ¿Cuál es la versión recomendable de Java?

Según la página web oficial de Java, deberías actualizar a el último parche (o versión) para el máximo rendimiento y seguridad.

5. ¿Qué es el JRE?

El JRE, Java Runtime Environment incluye Java Virtual Machine (JVM), las clases del núcleo de Java y bibliotecas. JRE representa la parte de tiempo de ejecución de Java, que es todo lo que necesita para ejecutar las aplicaciones de Java WebStart desde un navegador web. Sin embargo, no incorpora herramientas de desarrollo, dado que ya forman parte de Java Development Kit (JDK).

6. ¿Qué es el JVM?

El JVM (Java Virtial Machine) es una parte de Java que se encarga de ejecutar un programa o aplicación escrita en Java. Se incluye en la descarga de Java automáticamente
